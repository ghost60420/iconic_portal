# Costing Admin Guide

## Approve workflow
- Only users with costing approve permission can approve.
- Approved costings are locked from edits.
- Unlock requires a reason and is logged.

## Audit log
Actions logged:
- Created
- Updated
- Approved
- Unlocked
- Exported

Audit records include who, when, and changes to key fields.

## Data safety
- Do not delete line items from approved sheets.
- If corrections are needed, unlock with reason or create a new costing.
